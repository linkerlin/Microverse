### 7. 修复角色坐下后图片消失BUG ✅
- **根因**：`Chair.sit_character()` 的 Z-index 逻辑有误
  - `sit_direction == "up"`（BackChair）时，椅子 z_index 设为 1，角色设为 0
  - 结果：椅子精灵遮挡角色精灵，角色"消失"
  - FrontChair（sit_direction="down"）无此问题，因为角色 z_index=1 > 椅子 z_index=0
- **修复**：`Chair.gd` `sit_character()` 中简化 Z-index 逻辑——角色永远 z_index = base_z_index + 1，椅子 z_index = base_z_index
